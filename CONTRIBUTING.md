# 🤝 コントリビューションガイド

utip ライバー掲示板プロジェクトへのコントリビューションをありがとうございます！

## 📋 開発の流れ

1. **フォーク**: このリポジトリをフォーク
2. **ブランチ作成**: `git checkout -b feature/amazing-feature`
3. **開発**: 変更を加える
4. **コミット**: `git commit -m 'Add amazing feature'`
5. **プッシュ**: `git push origin feature/amazing-feature`
6. **プルリクエスト**: GitHubでプルリクエストを作成

## 🛠️ 開発環境のセットアップ

```bash
# リポジトリをクローン
git clone https://github.com/YOUR_USERNAME/utip-liverboard.git
cd utip-liverboard

# 依存関係をインストール
npm run install:all

# 開発サーバーを起動
npm run dev
```

## 📝 コーディング規約

- **JavaScript/TypeScript**: ESLintとPrettierを使用
- **コミットメッセージ**: [Conventional Commits](https://www.conventionalcommits.org/)に従う
- **ブランチ名**: `feature/`, `bugfix/`, `hotfix/` のプレフィックスを使用
